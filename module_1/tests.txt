1.1.1.	Тестовый сценарий «Добавление существующего товара в корзину».

Шаги воспроизведения: 
1. Зайти на главную сайта >> Все товары >> Выбрать любой товар. Проверка: загружается новая страница с выбранным товаром. 
2. Нажать клавишу «Добавить в корзину». Проверка: Появляется уведомление о том, что товар добавлен в корзину. 

1.2.1. Тестовый сценарий «Написание отзыва с некорректным адресом почты».

Шаги воспроизведения: 
1. Зайти на главную сайта >> Все товары >> Выбрать любой товар. Проверка: загружается новая страница с выбранным товаром.
2. Нажать клавишу «Написать отзыв». Проверка: загружается страница с формой отзыва. 
3. Ввести произвольные данные в форму.
4. В поле «Адрес электронной почты» ввести некорректный адрес без знака «@» >> Нажать «Сохранить отзыв». 
Проверка: возникает уведомление о некорректном адресе.  

2.1.1.	Тестовый сценарий «Регистрация с ненадежным паролем».

Шаги воспроизведения: 
1. Зайти на главную сайта >> "Войти или зарегистрироваться". Проверка: открывается новая страница с формой регистрации.
2. Ввести произвольный адрес электронной почты. 
3. В поле "Пароль" ввести ненадежный пароль, например, 1234. Проверка: Появляется уведомление о ненадежности пароля. 

2.2.1.	 Тестовый сценарий «Авторизация с неверным паролем».

Предусловие: наличие тестовой учетной записи. 
Шаги воспроизведения: 
1. Зайти на главную сайта >> "Войти или зарегистрироваться". Проверка: открывается новая страница с формой авторизации.
2. Ввести адрес электронной почты тестовой УЗ. 
3. В поле "Пароль" ввести неверный пароль. Проверка: Появляется уведомление о неверном пароле.
